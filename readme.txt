Entropy 2048

This is a simulator of the game 2048 in Lua, allowing to run repeated games
possibly with artificial players.
Original game: gabrielecirulli.github.io/2048/

License: GPL v3
Usage: lua entropy2048.lua [player_file [num_games]]

where player_file is the Lua module of an artificial player (without extension).
LuaJIT is recommended for better performance if you want to run lots of
games.
